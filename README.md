### Driver Risk Score Prediction App
This repository contains a Streamlit application for predicting driver risk scores based on the features: "Conviction_Points", "Geo_Risk", "Age", "Years_of_Experience", and "Vehicle_Type". The application allows users to choose either Regression or Classification models, upload training and testing data, train models, test models, and download the prediction outcomes.

### Deployment on GCP Cloud Run
The application has been deployed on Google Cloud Platform's Cloud Run service.
